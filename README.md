# Аудит Центр Плюс

Статический лендинг для GitHub Pages.

Планируемый адрес демо:

https://info14fourteen-creator.github.io/audit-center-plus-landing/

## Публикация

1. Загрузить файлы в GitHub-репозиторий.
2. В настройках репозитория открыть `Settings → Pages`.
3. Выбрать публикацию из ветки `master` или `main`, папка `/root`.
4. Сохранить настройки и дождаться URL GitHub Pages.

## Чат

Виджет чата уже работает в статическом режиме по базе сайта: консультирует, задает уточняющие вопросы и мягко ведет к заявке. Для подключения OpenAI нужен серверный proxy, чтобы API key не оказался в публичном JavaScript. Схема описана в [docs/chatbot-integration.md](docs/chatbot-integration.md).

## Cookies

На сайте есть уведомление о cookies. После согласия сохраняется технический session id, источник перехода, путь лендинга и черновик данных из AI-чата в локальном хранилище браузера.
